# ToolMatch
A curated marketplace with ROI scoring.

## Features
* Browse a curated list of ML tools with ROI scores, reviews, and categories
* Filter tools by category
* Sort tools by ROI score
* View detailed tool information, including user reviews and pricing info

## Usage
1. Create a `ToolMatch` instance
2. Add tools to the instance using the `add_tool` method
3. Filter tools by category using the `filter_by_category` method
4. Sort tools by ROI score using the `sort_by_roi` method
5. View detailed tool information using the `get_tool_details` method
