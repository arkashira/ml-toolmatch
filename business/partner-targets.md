# Partner Targets
## Introduction
The ml-toolmatch product aims to provide a comprehensive machine learning tooling marketplace or advisor service for Series A/B startups. To enhance the value proposition and user experience, strategic partnerships with relevant SaaS/API providers are crucial. This document outlines a partner integration roadmap, highlighting 7 key targets with rationale, free-tier limits, integration effort, and value-add.

## Partner Targets

1. **Hugging Face**
	* API: Hugging Face Hub API
	* Free-tier limits: 1000 requests/day
	* Integration effort: M
	* Value-add: Simplifies ML model deployment and management for users
	* Rationale: Hugging Face is a leading platform for ML model sharing and deployment, making it an ideal partner for ml-toolmatch
	* Affiliate/Revenue-share: Yes, Hugging Face offers a revenue-sharing program for partners
2. **Google Cloud AI Platform**
	* API: Google Cloud AI Platform API
	* Free-tier limits: $300 credit for new users
	* Integration effort: L
	* Value-add: Enables users to deploy and manage ML models on Google Cloud
	* Rationale: Google Cloud AI Platform is a popular choice for ML deployments, and integration with ml-toolmatch can enhance user experience
	* Affiliate/Revenue-share: Yes, Google Cloud offers a partner program with revenue-sharing opportunities
3. **AWS SageMaker**
	* API: AWS SageMaker API
	* Free-tier limits: 12 months free for new users
	* Integration effort: L
	* Value-add: Allows users to deploy and manage ML models on AWS
	* Rationale: AWS SageMaker is a widely used platform for ML deployments, and integration with ml-toolmatch can expand user options
	* Affiliate/Revenue-share: Yes, AWS offers a partner program with revenue-sharing opportunities
4. **Weights & Biases**
	* API: Weights & Biases API
	* Free-tier limits: 100,000 experiments/month
	* Integration effort: M
	* Value-add: Enhances ML experiment tracking and management for users
	* Rationale: Weights & Biases is a popular platform for ML experiment tracking, making it a suitable partner for ml-toolmatch
	* Affiliate/Revenue-share: Yes, Weights & Biases offers a revenue-sharing program for partners
5. **DataRobot**
	* API: DataRobot API
	* Free-tier limits: Custom pricing for enterprises
	* Integration effort: L
	* Value-add: Enables automated ML model development and deployment for users
	* Rationale: DataRobot is a leading platform for automated ML, and integration with ml-toolmatch can enhance user experience
	* Affiliate/Revenue-share: Yes, DataRobot offers a partner program with revenue-sharing opportunities
6. **Alibaba Cloud Machine Learning Platform**
	* API: Alibaba Cloud Machine Learning Platform API
	* Free-tier limits: $100 credit for new users
	* Integration effort: M
	* Value-add: Allows users to deploy and manage ML models on Alibaba Cloud
	* Rationale: Alibaba Cloud is a growing platform for ML deployments, and integration with ml-toolmatch can expand user options
	* Affiliate/Revenue-share: Yes, Alibaba Cloud offers a partner program with revenue-sharing opportunities
7. **ClearML**
	* API: ClearML API
	* Free-tier limits: 100,000 experiments/month
	* Integration effort: M
	* Value-add: Enhances ML workflow management and collaboration for users
	* Rationale: ClearML is a popular platform for ML workflow management, making it a suitable partner for ml-toolmatch
	* Affiliate/Revenue-share: Yes, ClearML offers a revenue-sharing program for partners

## Prioritization
The partner targets are prioritized based on the following criteria:

1. Affiliate/Revenue-share opportunities
2. Integration effort (lower effort preferred)
3. Value-add to the user
4. Market presence and popularity of the partner

By integrating with these 7 partner targets, ml-toolmatch can enhance the user experience, expand deployment options, and generate revenue through affiliate and revenue-sharing programs.
